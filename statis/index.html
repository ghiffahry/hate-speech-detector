About This Project

Sistem Deteksi Ujaran Kebencian Menggunakan Model Transformers

Project Overview

Sistem deteksi ujaran kebencian ini menggunakan teknologi deep learning terdepan dengan model IndoBERT yang telah dioptimalkan khusus untuk bahasa Indonesia. Sistem ini dapat menganalisis teks dengan akurasi 89.44% dan memberikan confidence interval untuk setiap prediksi dengan AUC score 96.16%.

Technology Stack

Python & FastAPI
Transformers & PyTorch
JavaScript & Chart.js
Docker & REST API

Key Features

  • Real-time hate speech detection with 89.44% accuracy
  • Batch processing capabilities up to 1000 texts
  • CSV file upload and processing
  • Statistical confidence intervals
  • Interactive data visualizations with Plotly
  • Word frequency analysis and text characteristics
  • Risk assessment and analytics dashboard
  • Export and download results in multiple formats

Model Information

Base Model: indolem/indobert-base-uncased
Training Data: Indonesian hate speech corpus (10,000+ samples)
Classes: Binary (Hate Speech / Safe)
Max Length: 512 tokens
Parameters: 124M parameters
AUC Score: 96.16%
